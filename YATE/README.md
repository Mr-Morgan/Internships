# изменения в версии от 31.05.2020

- добавлен виджет HexEdit для просмотра файлов любого размера в HEX и ASCII режиме ( открываются из контекстного меню файл-менеджера)

- переработан класс MenuBar (вставлена обработка открытия HEX файлов). Клас FileOpen теперь открывается только один для всех виджетов.

- преработан класс FileOpen. Теперь он для каждого виджета(textedit, hexedit) открывает сначала буфер. Потом открывается виджет в аргументах которого есть буфер.
	добавлен контейнер(вектор) в котором хранятся указатели на все открытые буферы виджетов. Добавлен Get метод для контейнера.

- переработан класс Buffer. Все действия с файлами перенесены сюда(loadFile, saveFile,..). При открытии файлов убраны флаги QFile:Text
	Добавлен метод getTextFragment для чтения заданных фрагментов файла(используется в виджете HexEdit)

- TextEdit переработан с учетом всех изменений. Максимальный размер для открытия файлов установлен в 100Мб.

Bugfixes:
- template_style.css  исправлен минимальный размер скролбара с 0 до 20
- в классе FileManager контектстное меню вызывается после MouseReleaseEvent вместо MousePressEvent
